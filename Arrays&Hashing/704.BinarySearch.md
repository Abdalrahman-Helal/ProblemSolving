# Binary Search After Sorting (C++)

## Code
```cpp
class Solution {
public:
    int search(vector<int>& nums, int target) {
        
        sort(nums.begin(), nums.end());
        return binarySearch(nums, target);
    }

    int binarySearch(vector<int>& arr, int x)
    {
        int low = 0;
        int high = arr.size() - 1;

        while (low <= high)
        {
            int mid = low + (high - low) / 2;

            if (arr[mid] == x)
            {
                return mid;
            }
            if (arr[mid] < x)
            {
                low = mid + 1;
            }
            else
            {
                high = mid - 1;
            }
        }
        return -1;
    }
};
```

---

## Explanation (Short)
1. First, the vector is sorted using `sort()`.
2. Then Binary Search is applied on the sorted array.
3. If the target exists, return its index.
4. Otherwise, return -1.

---

## Complexity
- Time Complexity:
  - Sorting: O(n log n)
  - Binary Search: O(log n)
  - Total: **O(n log n)**
- Space Complexity: O(1) (ignoring sort's internal space)

---

## Note
Sorting changes the original order of the array.  
So the returned index is the index in the **sorted array**, not the original array.
