# Contains Duplicate - C++ Solution

## Overview

This C++ program checks if a given array of integers contains any duplicates.  
It uses an `unordered_set` to store unique elements and compares its size with the original array.

---

## C++ Code

ðŸ”— **Problem URL:**  
https://leetcode.com/problems/contains-duplicate/description/

```cpp
#include <vector>
#include <unordered_set>
using namespace std;

class Solution {
public:
    bool hasDuplicate(vector<int>& nums) {
        unordered_set<int> us(nums.begin(), nums.end());
        if(us.size() != nums.size())
        {
            return true;
        }
        else
        {
            return false;
        }
    }
};
````

---

## How It Works

1. Insert all elements of `nums` into an `unordered_set`.
2. Compare the size of the set with the size of the original array:

   * If `us.size() != nums.size()`, there is at least one duplicate â†’ return `true`.
   * Otherwise â†’ return `false`.
