# Longest Consecutive Sequence - C++ Solution

## Problem Overview

This solution finds the length of the **longest consecutive elements sequence** in an unsorted integer array.

The algorithm runs in **O(n)** time using an `unordered_set`, making it optimal for this problem.

üîó **LeetCode Problem:**  
https://leetcode.com/problems/longest-consecutive-sequence/

---

## Approach

1. Store all elements in an `unordered_set` for **O(1)** average lookup.
2. Iterate through each unique number.
3. Only start counting when the current number is the **start of a sequence**  
   (i.e., `num - 1` does not exist in the set).
4. Expand the sequence forward and count its length.
5. Keep track of the maximum length found.

## C++ Code (LeetCode Compatible)

```cpp

// My Soultion
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {

        if(nums.size() == 0)
        {
            return 0;
        }
        unordered_set<int> uset;

        for(int i : nums)
        {
            uset.insert(i);
        }

        if(uset.size() == 1)
        {
            return 1;
        }

        vector<int> arr;
        for(int i : uset)
        {
            arr.push_back(i);
        }

        sort(arr.begin(), arr.end());

        int counter = 0;
        int res = 0;
        
        for(int i = 0; i < arr.size()-1; i++)
        {   
           if(i == 0)
           {
                counter = 1;
                if( (arr[i+1] - arr[i] == 1) )
                {
                    counter++;
                }
                else
                {
                    if(counter > res )
                    {
                        res = counter;
                    }
                    counter = 1; 
                }
           }

           else 
           {
                if((arr[i+1] - arr[i]) == 1)
                {
                    counter++;
                }
                else
                {
                   if(counter > res )
                    {
                        res = counter;
                    }
                    counter = 1;
                }
           }
        }

        if(counter > res )
        {
           return counter;

        }
        else
        {
            return res;
        }
    }
};


// Optimal Solution 
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_set<int> numSet(nums.begin(), nums.end());
        int longest = 0;

        for (int num : numSet) {
            // Check if num is the start of a sequence
            if (numSet.find(num - 1) == numSet.end()) {
                int length = 1;

                // Count consecutive numbers
                while (numSet.find(num + length) != numSet.end()) {
                    length++;
                }

                longest = max(longest, length);
            }
        }

        return longest;
    }
};
```

## Key Notes

- Each number is processed at most once.
- Avoids sorting, which would cost `O(n log n)`.
- Efficient and clean solution suitable for large inputs.
