# Concatenation of Array - C++ Solution

## Problem Overview

Given an integer array `nums` of length `n`, return an array `ans` of length `2n` where:

- `ans[i] == nums[i]`
- `ans[i + n] == nums[i]`

This means the array is concatenated with itself.

## Approach

1. Get the size of the input array.
2. Create a new array with size `2 * n`.
3. Copy each element of `nums` into:
   - the first half of `ans`
   - the second half of `ans`
4. Return the resulting array.

This approach is simple, efficient, and runs in linear time.

---

## C++ Code (NeetCode / LeetCode Compatible)

```cpp
class Solution {
public:
    vector<int> getConcatenation(vector<int>& nums) {
        int size = nums.size();
        vector<int> ans(size * 2);

        for (int i = 0; i < size; i++) {
            ans[i] = nums[i];
            ans[i + size] = nums[i];
        }

        return ans;
    }
};
```
