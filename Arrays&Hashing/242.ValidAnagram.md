# Valid Anagram - C++ Solution

## Problem Overview

Given two strings `s` and `t`, check if `t` is an anagram of `s`.  
An anagram means both strings contain the same characters with the same frequency.

---

## Approach 2: Using `unordered_map`

This solution uses two `unordered_map` to count character frequencies dynamically.

```cpp
#include <unordered_map>
#include <string>
using namespace std;

class Solution {
public:
    bool isAnagram(string s, string t) {
        unordered_map<char,int> um1;
        unordered_map<char,int> um2;

        if(s.size() != t.size()) {
            return false;
        }

        for(char i : s) {
            if(um1.find(i) == um1.end()) {
                um1.insert({i,1});
            } else {
                um1[i] = um1[i] + 1;
            }
        }

        for(char i : t) {
            if(um2.find(i) == um2.end()) {
                um2.insert({i,1});
            } else {
                um2[i] = um2[i] + 1;
            }
        }

        if(um1.size() != um2.size()) {
            return false;
        }

        for(char i : s) {
            if(um1[i] != um2[i]) {
                return false;
            }
        }

        return true;
    }
};
```

**Explanation:**

1. Check if string lengths are equal.
2. Count frequency of each character in both strings using `unordered_map`.
3. Compare the maps:

   * First check if sizes match.
   * Then check each character frequency.
4. Return `true` if all match, otherwise `false`.

✅ **Time Complexity:** O(n)
✅ **Space Complexity:** O(n) (for the maps)
