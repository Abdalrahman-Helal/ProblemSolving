# Product of Array Except Self

## üß© Problem Description
Given an integer array `nums`, return an array `answer` such that  
`answer[i]` is equal to the **product of all the elements of `nums` except `nums[i]`**.

### Constraints:
- Must run in `O(n)` time
- Do **not** use division
- Works correctly with zeros

https://leetcode.com/problems/product-of-array-except-self/

## üß† My Approach (Prefix & Suffix Products)

This approach uses **two auxiliary arrays**:
- `lr` ‚Üí prefix products (left to right)
- `rl` ‚Üí suffix products (right to left)

For each index `i`:
```

result[i] = product of elements before i √ó product of elements after i

````

---

## ‚úÖ My C++ Implementation

```cpp
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        vector<int> lr;
        vector<int> res;

        // Prefix product
        for(int i = 0; i < nums.size(); i++) {
            if(i == 0)
                lr.push_back(nums[i]);
            else
                lr.push_back(nums[i] * lr[i-1]);
        }

        // Suffix product
        vector<int> rl(lr.size());
        for(int i = nums.size() - 1; i >= 0; i--) {
            if(i == nums.size() - 1)
                rl[i] = nums[i];
            else
                rl[i] = nums[i] * rl[i+1];
        }

        // Final result
        for(int i = 0; i < nums.size(); i++) {
            int left  = (i > 0) ? lr[i-1] : 1;
            int right = (i < nums.size() - 1) ? rl[i+1] : 1;
            res.push_back(left * right);
        }

        return res;
    }
};
````

## üöÄ Optimal Approach (Constant Extra Space)

This solution:

* Uses the output array itself
* Computes prefix products in the first pass
* Multiplies suffix products in the second pass
* Requires **no extra arrays**

---

## ‚úÖ Optimal C++ Implementation

```cpp
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        vector<int> output(nums.size(), 1);

        int left = 1;
        for (int i = 0; i < nums.size(); i++) {
            output[i] *= left;
            left *= nums[i];
        }

        int right = 1;
        for (int i = nums.size() - 1; i >= 0; i--) {
            output[i] *= right;
            right *= nums[i];
        }

        return output;
    }
};
```
