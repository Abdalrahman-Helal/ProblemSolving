# Container With Most Water

**LeetCode Problem URL:** https://leetcode.com/problems/container-with-most-water/

## ğŸ§  Problem Description

You are given an integer array `height` where each element represents the height of a vertical line drawn on the x-axis.

- The index of the array represents the x-coordinate.
- The value at each index represents the height of a vertical line.

Choose **two different lines** such that together with the x-axis they form a container that can hold the **maximum amount of water**.

### âš ï¸ Important Notes
- You cannot tilt the container.
- The water amount is limited by the **shorter line**.

## ğŸ§¾ Code Implementation (C++)

```cpp
class Solution {
public:
    int maxArea(vector<int>& height) {
        int area = 0;
        int maxArea = 0;
        int l = 0;
        int r = height.size() - 1;

        while (l < r) {
            area = (r - l) * min(height[l], height[r]);
            maxArea = max(area, maxArea);

            if (height[l] > height[r]) {
                r--;
            } else {
                l++;
            }
        }

        return maxArea;
    }
};
```
---

## ğŸ“ How Area Is Calculated

For two indices `l` and `r`:

```
Area = (r - l) Ã— min(height[l], height[r]) 
```

---

## ğŸ’¡ Idea Behind the Solution (Two Pointers)

- Start with two pointers:
  - `l` at the beginning (index `0`)
  - `r` at the end (last index)
- Calculate the area between them.
- Update the maximum area found so far.
- Move the pointer that has the **smaller height**:
  - Because the smaller height is the limiting factor.

This guarantees an **O(n)** time complexity.
